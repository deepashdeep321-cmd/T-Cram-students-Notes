# 📚 T-Cram – Notes Sharing Platform for College Students

T-Cram is a web-based notes sharing platform built for college students to discover, upload, and organize subject-wise study materials — making collaborative learning simple and accessible.

## 🚀 Live Demo
> Deploy on Netlify and paste the link here

---

## ✨ Features

- 📂 **Subject-wise browsing** — Filter notes by Data Structures, DBMS, OS, OOP, ML, Networks
- 🔍 **Search** — Find notes by title, subject, or topic instantly
- ⬆️ **Upload Notes** — Share your notes with name, title, subject, and file
- 📥 **Download Counter** — See how many students downloaded each note
- 📱 **Responsive Design** — Works on mobile and desktop

## ⚙️ How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/t-cram-notes-platform.git
   ```

2. Open the folder and launch `index.html` in your browser — no server needed!

---

## 🌐 How to Deploy on Netlify

1. Go to [netlify.com](https://netlify.com) and sign up
2. Click **"Add new site"** → **"Deploy manually"**
3. Drag and drop the project folder
4. Your site goes live instantly with a public URL!
